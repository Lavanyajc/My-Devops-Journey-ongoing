### ğŸ”§ Commands:

```bash

```

### ğŸ“˜ Explanation:

| Command   | Purpose                                       |
| --------- | --------------------------------------------- |
| `pwd`     | Tells you the exact directory you are in      |

| `cd`      | Changes directories (`~` = home, `..` = back) |
| `sudo -i` | Gains root access (admin privileges)          |
| `exit`    | Exits from root back to regular user          |

| `rm`    | Deletes a file                          |
